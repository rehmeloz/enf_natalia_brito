# 📋 SPA para negócio local

SPA informativo para empreendedora de negócio local (Enfermagem & Estética)

## 🚀 Tecnologias Utilizadas

### FrontEnd
- **TypeScript** — Linguagem principal do FrontEnd
- **JavaScript** - Linguagem secundária do FrontEnd
- **React** - Biblioteca JavaScript utilizada na montagem de interfaces de usuário (UI)
- **Tailwind** - Framework CSS utilizado para estilização


## ✨ Funcionalidades

- **Visualização** de serviços e detalhes sobre o negócio local, possuindo seções que abordam diferentes informações, como Serviços, 
Video, Sobre mim, Depoimentos e Contato.
- **Botões de encaminhamento** direto para contato no WhatsApp e página do Instagram.
- **Utilização de Environment Variables** para proteção de dados sensíveis.
